# Supply_Chain_Game
## Interactive Semiconductor Supply Chain Disruption Simulation Game made by Raagini Chandra
I made this game using a CLI tool publically available on github which can be found here: https://github.com/noltron000-coursework/adventure
It created the decision tree based Choose Your Own Adventure Game with me entering my own prompts and then produced several python markdown files which I used an Markdown to HTML converter script to turn into html pages/files which I saved to my computer in vscode and uploaded to my github in its own repository with nothing else in the repository other than the files and then I made the repository available by using GitHub Pages. You can play this game at the following link: https://rchandra17.github.io/Supply_Chain_Game/
